# Mamani - Smart Finance Companion (Flutter scaffold)

This is a minimal Flutter scaffold created from the HTML designs in the repository. It contains three screens approximating the provided designs:

- Dashboard
- Analysis
- Log Expense

Prerequisites
- Flutter SDK (stable channel)

Run
```bash
cd flutter_app
flutter pub get
flutter run
```

Files
- `lib/main.dart` - entry point
- `lib/screens/*` - screen implementations
- `lib/models/sample_data.dart` - sample data used by screens

This scaffold is intentionally minimal and focused on UI structure. Replace placeholders and wire real data sources as needed.
